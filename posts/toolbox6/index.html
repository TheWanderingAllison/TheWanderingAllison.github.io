Windows 11 无法打开图片/视频/文本，错误：没有注册类

  • Toolbox
18 Jan, 2022

前几天电脑突然所有图片类文件都无法打开，双击之后错误提示如下：

不管是用 Google 还是 duckduckgo 搜索中文关键词得出的解决方案都不管用。真的蛮无语，中文搜索结果已经差成这样了，很多都是几年前的帖子，或者是垃圾网站骗点击的无效页面。

换成英文搜 can’t open pictures Class not registered ，点开排名靠前的这篇 # Fix Class not registered error when opening pictures in Windows 11/10 看了下，给了几个可能解决方案：

  1. Run Windows Store Apps troubleshooter
  2. Run SFC scan
  3. Check your default app to view photos
  4. Re-register or reinstall the Photos app via Windows PowerShell

前三种对我的电脑都无效。第四种需要在 Microsoft Store 里先把 Microsoft Photos app 更新到最新版，接着去设置-应用- Microsoft Photos -高级选项，重置或者修复都试一遍（我忘了是哪个起作用了），成功！

另， txt 和视频类文件出现“没有注册类”错误，依然可遵循以上步骤解决，只是第四种方法里的软件对应地要变成 Notepad 或 Movies & TV。

过去都说百度的结果不靠谱，但现在事实是，整个中文生态里的搜索结果都不靠谱。或许也存在一些中文的优质内容，但它们都被困在平台自己的生态里，外人进不去，内容出不来。现在我无论搜什么，都只能用我的三脚猫英语想想英文关键词，通常 Google 出的结果前几页，甚至是前几条就能解决问题。

这也是为什么我把这么简单的问题水了篇 blog 的原因。没准就有人搜相关中文，可能、或许，会看到我这条吧。
